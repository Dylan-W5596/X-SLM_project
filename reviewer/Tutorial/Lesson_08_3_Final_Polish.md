# 第 8.3 課：動態背景與現代過渡 (Transitions) - 完結篇

這節課我們解析 App 畫面的最後一層面紗：**動態視覺與過渡細節**。

---

## 1. 動態 Dither 背景 (藝術與代碼)

本專案使用了一種稱為 **Dither (抖動)** 的復古藝術特效。
*   **作法**：透過 CSS 的 `repeating-conic-gradient` 或者是 Shader（著色器）語法。
*   **優點**：它不需要載入沉重的影片。它只是在畫布上不斷跑幾行數學公式，就能產生無限變幻的波紋。這能讓 App 看起來有一種「生命力」。

---

## 2. 轉場動畫：Cubic Bezier 的魔力

在 CSS 中，我們沒有寫 `transition: 0.3s linear`，而是：
`transition: all 0.5s cubic-bezier(0.4, 0, 0.2, 1);`

### 深度解析：
*   **Linear (線性)**：動作很死板，像機器人。
*   **Cubic Bezier (貝茲曲線)**：這能模擬物理慣性：**加速啟動、緩速停下**。這是讓介面感覺「順滑」像高級車一樣的關鍵參數。

---

## 3. 響應式佈局的極限 (Flex & Grid)

請看對話氣泡的排版：
```css
max-width: 85%;
align-self: flex-end; /* 自己在右邊 */
```
我們使用了 Flexbox 的靈活性。當使用者縮放視窗時，氣泡會自動調整寬度，字體也不會溢出。

---

## 4. 終章：整合的力量

從 Phase 1 到 Phase 8，您會發現一件重要的事情：
**好的產品不是只有 AI。**
AI 只是邏輯（後端），資料庫是記憶（底層），React 是骨肉（前端），而 Electron 是它能跑在各個電腦上的靈魂（平台）。所有的技術環節緊扣在一起，才造就了您眼前的這台 **Llama Electron**。
